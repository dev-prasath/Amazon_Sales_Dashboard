# 🛒 Amazon India: A Decade of Sales Analytics 📊🇮🇳

## 📌 Project Overview
This project is an end-to-end e-commerce analytics solution built using Amazon India’s 10-year transactional dataset (2015–2025). It transforms raw, messy data into meaningful business insights through data cleaning, analysis, and visualization.

---

## 🚀 Key Features
- 🧹 Advanced Data Cleaning (handling missing values, inconsistencies, outliers)
- 📊 Exploratory Data Analysis (20+ visualizations)
- 🗄️ SQL Database Integration
- 📈 Interactive Dashboards (Power BI / Streamlit)
- 💡 Business Insights for decision-making

---

## 🧠 Business Use Cases
- Revenue trend analysis & growth forecasting  
- Customer segmentation (RFM Analysis)  
- Product & category performance analysis  
- Geographic sales insights across Indian cities  
- Payment method evolution (UPI growth, COD decline)  
- Delivery performance & return analysis  

---

## 🛠️ Tech Stack
- **Python** (Pandas, NumPy)
- **Visualization**: Matplotlib, Seaborn  
- **Database**: SQL  
- **Dashboarding**: Power BI / Streamlit  
- **Other**: Data Cleaning, Statistical Analysis  

---


## 📊 Dataset
The dataset contains ~1,000,000 transactions from 2015–2025 with realistic e-commerce patterns.
